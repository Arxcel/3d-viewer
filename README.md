## Requirements

Mac OS Sierra / High Sierra

## Installation

```bash
git clone https://github.com/Arxcel/basic_openGl_3dviewer.git ~/3dviewer
cd ~/3dviewer
make
```

## Manual
Translation :
```
    W,S,A,D,Q,E
```
Rotation :
```
    I,K,J,L,U,O
```
Show histogram:
```
    Z
```
